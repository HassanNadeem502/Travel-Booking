# Travel Booking Page

This project is a simple and responsive **Travel Booking Page** built using **HTML & CSS**.  
It provides a clean and user-friendly interface for users to book their travel by entering:

- Destination
- Check-in Date
- Check-out Date
- Number of Guests

## Features

- Responsive and modern design
- Easy-to-use booking form
- Basic HTML5 form validation
- Clean and organized code structure

## Technologies Used

- HTML5
- CSS3

## Files Included

- `index.html` — Main HTML file for the booking page
- `travel.css` — Stylesheet containing all CSS styles
